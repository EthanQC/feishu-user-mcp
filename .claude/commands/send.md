以用户身份发送飞书消息。

## 参数
- $ARGUMENTS: 收件人和消息内容，格式："收件人名称: 消息内容"

## 执行步骤

1. 解析 $ARGUMENTS，分离收件人名称和消息内容（以第一个冒号分隔）
2. 使用 `feishu-user-mcp` 的 `send_to_user` 工具，传入 `user_name` 和 `text`
3. 如果发送成功，告知用户消息已发送
4. 如果用户未找到，列出搜索结果供用户选择

## 注意
- 消息以你的个人飞书身份发送，不是机器人
- 支持发送给个人（单聊）和群组
- 如果收件人是群组，使用 `send_as_user` 工具并传入群的 chat_id
